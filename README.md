Just some useful stand alone web pages


Generate-Resources.html - Generates a random set of resources, which can be used for exercises
Net-Control-log - Just what it says, roboust net net logger which builds the roster as you go making repeat nets easy.  QRZ lookup, must have subscription.
Winlink-postion-report - Uses a generated .pos aprs file from winlink to look for sitreps.
Failure-generator - random failures for SET or other radio exercises.
Grid-to-gps - converts grid square to GPS coordinates and provide google maps link
qso-generator - use any image to create a qsl card
Welfare-RRI-star-wars - welfare requests using star wars characters for SET exercies.
